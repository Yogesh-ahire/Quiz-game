Quiz Buddy is a **fully interactive quiz platform** built using HTML, CSS, and JavaScript, featuring a **custom-built quiz engine**, real-time scoring, timer-based gameplay, and persistent user history.

The project focuses on **logic-driven design, API integration, and client-side state management**, making it a strong demonstration of frontend + system logic development.

# Features

### Custom Quiz Engine
A dynamic quiz engine that manages the complete quiz lifecycle:
- Question randomization  
- Answer validation  
- Score calculation  
- Progress tracking  

---

### API Integration (Open Trivia DB)
- Dynamic question fetching based on category and difficulty  
- Real-time data transformation and rendering  
- Supports multiple categories and quiz configurations  

---

### Timer-Based Gameplay
- Countdown timer for each question  
- Auto-skip functionality on timeout  
- Adds real-time pressure and engagement  

---

### Quiz Customization
- Select category  
- Choose difficulty (Easy, Medium, Hard)  
- Set number of questions  
- Optional timer control  

---

### Leaderboard System
- Tracks best scores per player  
- Ranking based on:
  - Highest score  
  - Fastest completion time  
- Displays top performers  

---

### Quiz History Tracking
- Stores previous attempts using localStorage  
- Shows:
  - Category  
  - Difficulty  
  - Score  
  - Time taken  
  - Date  

---

### Interactive UI / UX
- Clean and modern interface  
- Fully responsive design (mobile + desktop)  
- Smooth animations and transitions  
- Glassmorphism-style design elements

# Tech Stack

### Frontend
- HTML5  
- CSS3 (Responsive + Animations)  
- JavaScript (ES6+)  

---

### Core Logic
- Custom Quiz Engine  
- Timer-based event system  
- Randomized question selection  
- Score & progress calculation  

---

### Data Handling
- Open Trivia DB API  
- LocalStorage (client-side persistence)  

---

# System Design Overview

- Event-driven quiz flow (start → question → answer → result)  
- Modular functions for quiz lifecycle management  
- Dynamic DOM rendering based on user interaction  
- Client-side state persistence using localStorage  
- Timer-based control system for gameplay  

---

# How to Play
- Enter your name to start
- Choose a category or customize quiz
- Answer questions within the given time
- Earn points for correct answers

### Gameplay Rules
- Each correct answer gives points
- Timer auto-skips unanswered questions
- Final score is calculated at the end
